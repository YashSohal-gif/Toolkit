How it works

Each PDF page is rendered as a high-resolution image (via pdf.js) and placed full-bleed onto its own slide (via PptxGenJS) — entirely in your browser.

Is my file uploaded anywhere?

No — everything happens locally. Your PDF never leaves your device.

Can I edit the text after converting?

Not directly — since each slide is an image, you'd need to add a text box over it in PowerPoint if you want to edit specific words. For fully editable slides, you'd need the original presentation file.
