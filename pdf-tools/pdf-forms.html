About fillable forms

What if my PDF has no fields?

Not every PDF is a true fillable form — if none are detected, you'll see a message saying so. In that case, try the Edit PDF tool to add text boxes manually instead.

What field types are supported?

Text fields, checkboxes, radio button groups, and dropdown/choice fields are all detected and fillable.

Is my file uploaded anywhere?

No — form filling happens entirely in your browser. Your PDF never leaves your device.
