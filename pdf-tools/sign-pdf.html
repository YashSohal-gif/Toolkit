About signing

Is this a legally binding e-signature?

This places a visual signature image onto the page — it's great for informal signing, but it does not add cryptographic signature verification like some paid e-signature services.

Can I make a reusable signature file instead?

Yes — try the standalone Signature Maker tool to save a transparent PNG you can reuse anywhere.

Is my file uploaded anywhere?

No — signing happens entirely in your browser. Your PDF never leaves your device.
