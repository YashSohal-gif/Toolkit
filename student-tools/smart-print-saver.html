Tips for students

How many pages per sheet should I pick?

Each option stacks pages as full-width horizontal bands, one below the other, so the top of the sheet is always page 1: pick 2-3 for larger, easy-to-read text, or 4-5 to fit more pages per sheet and save the most paper.

When should I use black & white or invert?

Black & white removes color ink usage entirely — useful if your printer is set to color by default. Invert flips dark backgrounds to light, which is handy for slide decks or notes designed with a dark theme that would otherwise waste ink covering the whole page.

Is the text still readable after compression?

Yes — compression reduces file size for faster downloads and printing, but layout and page content stay exactly as they were, just smaller per page.

Want to make a foldable booklet instead?

Use the Booklet Maker tool — it arranges pages in the correct order for double-sided folded printing, with step-by-step instructions for your printer.
