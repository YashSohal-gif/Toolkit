How this works

4.0 scale grade points

A = 4.0, A- = 3.7, B+ = 3.3, B = 3.0, B- = 2.7, C+ = 2.3, C = 2.0, C- = 1.7, D+ = 1.3, D = 1.0, F = 0.0.

10-point scale grade points

Corresponds to common grade bands out of 10 (O=10, A+=9, A=8, B+=7, B=6, C=5, P=4, F=0) — check your institution's exact conversion since these can vary.

Is this the same as my official transcript GPA?

This gives you a close estimate using common grade-point conventions. Some institutions weight courses differently (e.g. excluding pass/fail or repeated courses), so always confirm with your registrar for official records.
